# MNGA ロゴ使用ポリシー

**Make Nippon Great Again（以下「本団体」）** のロゴおよびブランドアセットを使用するすべての方を対象としたガイドラインです。
本ポリシーは、本団体のブランドの一貫性・信頼性・尊厳を守るために定められています。

最終更新日：2026年4月24日

---

## 1. 基本方針

MNGA ロゴは、本団体の理念・活動を象徴する重要な資産です。
ロゴを使用する際は、以下のルールに従い、ブランドイメージを損なわない方法でご利用ください。

ロゴの使用には、原則として **本団体からの事前承認** が必要です。
個人ブログ・SNS等での非商用引用については本ポリシーの範囲内であれば事前承認は不要です。

---

## 2. ロゴの種類（バリエーション）

本団体のロゴは、以下の2種類を正式なロゴとして提供しています。

### 2.1 シンボルロゴ（正方形）
赤い円形の筆致内に白抜きで「MNGA」と表記された形式。
- 用途：アイコン、SNSプロフィール、小スペースでの単独使用
- ファイル：`mnga-logo.png`

### 2.2 横ロゴ（ヨコ型）
シンボルマークと文字を横並びに配置した形式。
- 用途：Webサイトヘッダー、書類、プレゼン資料の見出し
- ファイル：`MNGA_ヨコ.png`

> **注意：** 上記2種類以外の改変・再構成したロゴ（例：マークだけ別配置にしたもの）の使用は禁止です。

---

## 3. クリアスペース（余白）

ロゴの周囲には、視認性を保つため十分な余白（クリアスペース）を確保してください。

- **最小余白**：ロゴの「M」の文字の高さ分の余白を、ロゴの上下左右に確保すること
- 余白内に他のロゴ・テキスト・イラスト等を配置しないでください

```
┌─────────────────────────┐
│  ← 余白（M の高さ分）     │
│  ┌───────────────────┐ │
│  │                   │ │
│  │      MNGA ロゴ    │ │
│  │                   │ │
│  └───────────────────┘ │
│  ← 余白（M の高さ分）     │
└─────────────────────────┘
```

---

## 4. 最小使用サイズ

視認性を確保するため、以下のサイズを下回って使用しないでください。

| 用途 | シンボルロゴ | 横ロゴ |
|---|---|---|
| デジタル（Web、画面表示） | 32 × 32 px | 高さ 24 px 以上 |
| 印刷物 | 10 × 10 mm | 高さ 8 mm 以上 |

本団体のブランドカラーは以下です。ロゴおよびブランド表現で使用する色は原則これに準拠してください。

| 色名 | HEX | RGB | 用途 |
|---|---|---|---|
| MNGA レッド | `#c8392b` | 200, 57, 43 | プライマリカラー（ロゴ・強調） |

## 6. 背景での使用

ロゴを配置する背景は、十分なコントラストを確保してください。

### 推奨される背景
- **白背景**：シンボルロゴ・横ロゴの両方を最も推奨
- **淡色背景**（ライトグレー、オフホワイト等）：可
- **写真背景**：白系または無地に近い領域に限り可（顔・複雑な柄の上は不可）

### 推奨されない背景
- 赤系の背景（ロゴ自体が赤のため埋もれます）
- 高彩度の背景（コントラスト不足）
- ロゴが視認しづらい複雑な写真・パターン

---

## 7. 禁止事項

以下の使用方法は本ポリシー違反となり、いかなる場合も認められません。

| # | 禁止内容 |
|---|---|
| 1 | ロゴの**色を変更**すること（赤を別の色に変える等） |
| 2 | ロゴを**変形・伸縮**すること（縦伸ばし・横伸ばし） |
| 3 | ロゴに**影・グロー・3D効果**等のエフェクトを追加すること |
| 4 | ロゴを**回転・反転**して使用すること |
| 5 | ロゴの**一部分だけ**を切り出して使用すること |
| 6 | ロゴ内の**文字や配置を変更**すること |
| 7 | ロゴを**他のロゴや文字と結合**して新しいマークを作ること |
| 8 | ロゴを**透かし（ウォーターマーク）**として無断で使用すること |
| 9 | 本団体の**承認なくグッズ・印刷物**に使用すること |
| 10 | **公序良俗・本団体の理念に反する文脈**で使用すること |
| 11 | ロゴを**装飾の一部・フォントの代用**として文章中に組み込むこと |

## 8. コブランディング（他ブランドとの併用）

他団体・他企業のロゴと並べて表示する場合は、以下を守ってください。

- 各ロゴの間に **MNGAロゴの「M」の文字幅以上** のスペースを確保
- 各ロゴの **視覚的サイズを揃える**（幾何学的サイズではなく、視覚的に同等のサイズ）
- ロゴ間に区切り線を入れる場合は細い縦線（1〜2px）に限る
- 共催・後援・協賛など、関係性が誤解を招かないよう **役割表記を併記**すること

事前に本団体に共催・協賛関係の確認を行ってください。

## 11. 違反した場合

本ポリシーに違反するロゴ使用が確認された場合、本団体は以下の措置を取ることがあります。

1. 使用停止の要請
2. 該当ロゴ・コンテンツの削除要請
3. 法的措置（商標権・著作権に基づく）
